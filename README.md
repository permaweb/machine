# The Machine

To deploy a collection to the permaweb, all you need is a directory with the following:

- Your asset files
- A `collection.json` file that describes your project
- OPTIONAL - banner.png file to provide a banner for your collection

```
- assets
  - 0.png
  - 1.png
  - 2.png
  - collection.json
  - banner.png
```
